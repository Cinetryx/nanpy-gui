=========
nanpy-gui
=========

simple TraitsUI_ based GUI for Nanpy_

Installation
============

 * install Python_
 * install pip_
 * install Nanpy_
 * install TraitsUI_
 
Ubuntu
------
::

    sudo apt-get install python-pip python-traitsui python-traits
    sudo pip install nanpy
	

Run
---
::

    python -m nanpygui.run
    # or
    python  nanpygui/run.py
